Instruções p/ baixar o projeto:

1 - Baixar repositorio git : https://github.com/gmascena/testjavaimportfile.git

2 - Abrir o projeto "teste" no STS.


Instruções para rodar o teste:

1 - Copiar o arquivo CSV pretendido na pasta raiz do projeto. (Já há o arquivo-base na pasta)

2 - Executar RUN AS/Spring Boot App (O serviço será inicializado e executará a importação do arquivo)

3 - Executar o GET: http://localhost:8080/movies/1
